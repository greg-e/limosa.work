# Typography

## Font stack

Primary stack:

`"Avenir Next", "Helvetica Neue", Arial, sans-serif`

Fallback if unavailable:

`system-ui, -apple-system, "Segoe UI", Roboto, Arial, sans-serif`

## Hierarchy

- **H1**: 48–72px, weight 700, tight tracking
- **H2**: 28–40px, weight 700
- **H3**: 20–26px, weight 600–700
- **Body Large**: 20–22px, weight 400, line-height 1.4–1.5
- **Body**: 16–18px, weight 400, line-height 1.5
- **Meta / Labels**: 12–14px, weight 500–600, increased letter spacing

## Typography principles

- Prioritize legibility over style.
- Use strong contrast and ample spacing.
- Keep paragraph width readable (roughly 60–90 chars).
- Reserve all-caps for short labels only.
