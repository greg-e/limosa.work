# Actual Brand Identity Package

This package defines the foundational brand system for Actual and provides ready-to-use visual assets.

## What this includes

- **Brand guide**: positioning, personality, design principles
- **Color palette**: primary, secondary, neutrals, semantic usage
- **Typography**: hierarchy and practical fallback stack
- **Messaging**: value proposition, elevator pitch, proof framing
- **Usage checklist**: fast QA before publishing materials
- **Logo assets** in `assets/brand/`
- **Asset preview page**: `docs/brand-identity/asset-preview.html`

## Asset index

- `assets/brand/actual-wordmark-primary.svg` — primary logo for light backgrounds
- `assets/brand/actual-wordmark-light.svg` — reverse logo for dark backgrounds
- `assets/brand/actual-mark.svg` — icon mark (no wordmark)
- `assets/brand/actual-mark-light.svg` — light icon mark for dark surfaces

## Quick usage rules

1. Prefer the primary wordmark in most partner materials.
2. Keep clear space around logos equal to the mark diameter.
3. Do not recolor logos outside approved palette.
4. Do not stretch, rotate, or add shadows to logo assets.
5. Keep messaging grounded in operational truth, not hype.
