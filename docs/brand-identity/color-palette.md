# Color Palette

## Primary palette

| Token | Hex | Use |
|---|---|---|
| Ink | `#111111` | Core text, primary icons |
| Field Deep Green | `#1E4A42` | Primary brand color, key highlights |
| Earth Accent | `#CB6735` | Calls to action, status emphasis |
| Paper | `#F4EFE4` | Primary background |
| Stone | `#5C5850` | Secondary text |

## Supporting neutrals

| Token | Hex | Use |
|---|---|---|
| Mist | `#E8E4DC` | Dividers, subtle surfaces |
| Clay | `#9A9488` | Labels, metadata |
| White | `#FFFFFF` | Reverse surfaces |

## Semantic status colors

| Semantic | Hex | Use |
|---|---|---|
| Success | `#6BAB78` | Healthy/confirmed state |
| Warning | `#E0A040` | Needs attention |
| Info | `#4A90D0` | Cloud/system signal |

## Usage rules

- Keep contrast high for readability in field contexts.
- Use Earth Accent sparingly for emphasis, not decoration.
- Avoid introducing unapproved saturation-heavy colors.
- Do not gradient-fill logos.
