# Brand Guide

## Brand core

- **Brand name:** Actual
- **Category:** Field truth infrastructure for service operations
- **Promise:** Replace self-reported activity with machine-observed reality
- **Positioning:** Durable, practical, and operationally honest

## Design principles

1. **Evidence over aesthetics** — clear, measurable, believable
2. **Utility over ornament** — every visual element has a job
3. **Field-grade simplicity** — robust and legible in rough contexts
4. **Calm confidence** — never loud, never gimmicky

## Brand personality

- Grounded
- Precise
- Accountable
- Direct
- Trustworthy

## Visual language

- High-contrast, low-noise layouts
- Neutral base colors + one action accent
- Structured spacing and predictable hierarchy
- Diagrams and visuals should reveal system logic

## Brand voice

- Use short, factual statements
- Emphasize outcomes and operational clarity
- Avoid buzzwords and exaggerated claims
- Prefer “what happened” language over “what might happen”

## Audience framing

- **Owner / Operator:** certainty, accountability, route and labor clarity
- **Field Manager:** dispatch confidence, faster exception handling
- **Crew:** minimal friction, no extra app/admin burden

## Narrative anchor

> Actual turns field activity into verified operational truth.
